Welcome to the North End Makerspace file share
-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*-*

This share can be accessed by any computer in the makerspace.  Use
it to move files between computers in the space.

Rules:
------

    1. Do not delete files that don't belong to you.
    2. Do not store sensitive content here.  *Everyone* can see all the files.
    3. Create a folder with your first name and last initial, or something
       we can use to identify folder owners in case we need to do cleanup
       and want to check with you first.
    4. If you have any trouble, reach out in Discord.
